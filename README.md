# Front Dev Tool

## Example
```shell
npm start
```
then open browser   
http://127.0.0.1:8080/?sp=.abc,background-color,red&sp=.def,width,100,px&gv=globalVar,hello

## Query String Options
### sp: style property
* sp=selector, property, value[, type(px | %)] 

### gv: global variable
* gv=variable, value[, type(number)]
